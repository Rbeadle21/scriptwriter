# ScriptWriter — Stage Play Editor

A free, browser-based editor for writing stage plays and theatre scripts. No account, no downloads, no software to install — just open and start writing.

🔗 **[Open the Editor](https://rbeadle21.github.io/scriptwriter/)**

---

## Features

- **Script Editor** — Write your play using standard theatrical formatting: Acts, Scenes, Character dialogue, and Stage directions
- **Actor Zone** — A dedicated view for individual actors, with their lines highlighted
- **Screenplay Mode** — Switch to a screenplay-style layout
- **Cast List** — Manage your characters with descriptions in one place
- **Auto-save** — Your work is saved automatically in your browser
- **Print / Export to PDF** — Print your script or save it as a PDF directly from your browser
- **OpenDyslexic font** — Built-in dyslexia-friendly font option, works fully offline
- **Keyboard shortcuts** — Tab and Enter keys for quick block navigation

---

## How to Use

1. Visit the editor using the link above
2. Enter your play title and author name at the top
3. Use the sidebar buttons (or keyboard shortcuts) to add Acts, Scenes, Dialogue, and Stage Directions
4. Your script saves automatically in your browser
5. When ready, click **Print / PDF** to export

---

## Technical Notes

- Single HTML file — no dependencies, no build process
- Works offline once loaded
- All data is stored locally in your browser (nothing is sent to a server)
